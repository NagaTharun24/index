# Tharun Bandi - ServiceNow Developer Portfolio Website

## Overview

A professional, responsive, and modern portfolio website for Tharun Bandi - Associate Technical Consultant at Kaptius. This portfolio showcases expertise in ServiceNow development with 3+ years of experience.

## Features

### 🎯 Key Sections

- **Hero Section**: Professional introduction with profile image and quick action buttons
- **About Me**: Professional summary and career objective
- **Experience**: Timeline-based work experience display
- **Education**: Academic qualifications
- **Certifications**: ServiceNow certifications (CSA, CAD, ITSM, CIS)
- **Skills**: Comprehensive skills showcase with progress bars
  - ServiceNow Skills (ITSM, HRSD, CSM, Flow Designer, etc.)
  - Technical Skills (JavaScript, HTML5, CSS3, React, SQL, Git)
- **Projects**: Key projects worked on (CATS, NewCleo, Element, RM Education, Spectris, ICT Services)
- **Achievements**: Recognition and awards
- **Quick Links**: Resume download, GitHub, LinkedIn
- **Contact**: Contact information and contact form

### ✨ Special Features

- **Interactive Profile Modal**: Click profile image to see contact details
- **Smooth Animations**: CSS animations and transitions throughout
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Color Scheme**: Professional dark blue theme with accent colors
- **Modern UI**: Clean, elegant, and professional design
- **Sticky Navigation**: Easy navigation with active section highlighting
- **Form Validation**: Contact form with validation

## How to Use

1. **Download the files** to your device
2. **Open `portfolio.html`** in your web browser
3. **Customize content** by replacing placeholders with actual information
4. **Update images** by replacing the profile image URL
5. **Add your resume** PDF file in the project folder
6. **Deploy** to GitHub Pages, Netlify, or any web hosting service
